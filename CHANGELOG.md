
# Changelog

All notable changes are documented here.

## [Unreleased]

### Added
- Modular command system (`commands/` package)
- Cross-platform TTS via `utils/text_to_speech.py`
- Conversation memory for context-aware AI replies
- Weather lookup without an API key (Open-Meteo)
- Browser-based web search
- Note-taking command (save and read)
- Reminder command with threading timers
- Wake-word detection utility
- Text-mode CLI (`cli.py`)
- Unit tests with pytest
- GitHub Actions CI pipeline
- Logging to daily log files
- Environment variable configuration via `.env`

## [0.1.0]

### Added
- Initial voice assistant with speech recognition and Ollama integration
