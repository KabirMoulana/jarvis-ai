## What is this?

J.A.R.V.I.S. is a voice-activated AI assistant built to replicate Tony Stark's AI. It runs **fully locally**, responds to "Hey Jarvis", and has 109+ skills covering everything from smart home control to crypto prices.

---

## Quick Start

```bash
git clone https://github.com/KabirMoulana/jarvis-ai.git
cd jarvis-ai
pip install -r requirements.txt
ollama pull llama3.2        # https://ollama.com
cp .env.example .env
python main.py
```
