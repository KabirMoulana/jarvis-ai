## What is this?

J.A.R.V.I.S. is a voice-activated AI assistant built to replicate Tony Stark's AI. Runs **fully locally**, responds to "Hey Jarvis", and has **165+ skills** across 20+ categories.

---

## Quick Start

```bash
git clone https://github.com/KabirMoulana/jarvis-ai.git
cd jarvis-ai
pip install -r requirements.txt
ollama pull llama3.2
cp .env.example .env
python main.py
```
